# バランスモデル

## 現在値

- プレイ時間: 60秒
- 盤面: 6列×7行
- 初期デッキ: `ぐんまけん`, `だるま`, `あかぎさん`
- 文字プール: 13文字
- スコア: `30 × 文字数^3`
- 今日の差し入れ: 1日1回、Gだるま+1
- デイリースコア目標: 5000点でGだるま+1
- 週替わりチャレンジ: 日替わり同様にお題が変わる。20プレイ相当の回数/語数/合計スコアでGだるま+10
- リフレッシュ: 5秒クールダウン、減点なし
- スコアアップ: 8秒間 +30%
- 特殊アイテム: 現行版では出現なし
- 凸補正: 1凸ごとにスキル効果 +2%、最大10凸

## 初期デッキスコア

- `だるま`: 3文字、810点
- `ぐんまけん`: 5文字、3750点
- `あかぎさん`: 5文字、3750点

## 文字数差の意図

- 3文字デッキは完成頻度とコンボで強くなりやすいため、1語あたりの基礎点は控えめにする。
- 5文字以上は盤面調整と完成難度が上がるため、基礎点の伸びを大きくして、長い言葉を狙う理由を作る。
- 旧式 `100 × 文字数^2` は3文字900点、5文字2500点で、短い言葉の連鎖が総合スコアで勝ちやすかった。

## 調整優先度

1. 初回60秒で1回以上消えるか。
2. 3プレイ目で3回以上消えるか。
3. スライド操作に考える余地があるか。
4. 短いカードだけが強くなりすぎていないか。
5. 長いカードが気持ちよく高得点になっているか。
6. 今日の差し入れが復帰理由になり、日課やスタミナの圧を強めすぎていないか。
