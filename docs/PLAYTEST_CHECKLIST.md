# プレイテストチェックリスト

## 事前設定

- 端末: スマホ縦持ち想定、390px幅相当でも確認する。
- デッキ: `ぐんまけん`, `だるま`, `あかぎさん`。
- 1セッション: 1人あたり5プレイ以上。
- 記録: スコア、完成単語数、リフレッシュ使用数、スキル発動数。
- 記録: デイリー目標とデイリースコア目標の達成率、達成までの平均プレイ数、Gだるま+1報酬が嬉しいか。
- 記録: 次回ログイン時の今日の差し入れモーダルを受け取ったか、毎日開く理由として嬉しいか、プレイ直後の再挑戦やデッキ変更を邪魔していないか。
- 記録: 今日のことばを見てから遊んだか、狙いの言葉や話題作りに役立ったか。

## 観察ポイント

- 保存なしの初回トップメニューで、日課/スコア目標よりも `FIRST PLAY` とデッキが目に入るか。
- 初回プレイ開始後、`FIRST CLEAR` 案内が読めて、光る2枚のスライド操作が理解できるか。
- 初回で何も操作しない場合、8秒後のお手本で `ぐんまけん` が完成し、20秒以内に1回成功体験が出るか。
- チュートリアル後から初回完走までは、日課/スコア目標/今日のことば/今日の差し入れが表示されず、PLAYボタンとデッキの主導線が残っているか。
- 初回完走直後は今日の差し入れが出ず、再挑戦、デッキ構築、パック確認へすぐ進めるか。次回ログインでだけ差し入れモーダルが出て「無料の小さな復帰報酬」として読めるか。
- 初回完走後のミッション画面で、今日のことばが短く読め、図鑑ボタンで該当カードのメモへ自然に移動できるか。
- 3完走後のミッション画面で、日課/スコア目標の達成済み表示が「今日の区切り」として気持ちよく見えるか。
- 再起動後のトップメニューで `LAST RUN` が前回スコアと見どころワードだけを短く見せ、主ボタンへの導線を邪魔していないか。
- 通常ホームで `LAST RUN` の直後にゲーム/デッキ/パック/ミッション/設定へ触れ、今日のことばや達成カードがプレイ直後の反応導線を邪魔していないか。
- 通常ホームで `ゲーム/デッキ/パック/ミッション` の4つが横並びの主導線として読め、設定は右上の補助操作として十分見つけられるか。
- タイトル画面で基本カード3枚が豪華に見え、カード収集ゲームとしての期待感が出ているか。画面のどこを押しても進め、下部の `TOUCH START` がボタン風に見えすぎず、横スクロールや文字重なりがないか。
- 3完走後のミッション画面で、週替わり祭りチャレンジが「今週もう少し遊ぶ理由」として読め、日課より重く見えすぎないか。
- パック画面とスタミナ回復が、閉鎖テスト用の`TEST`表示として読め、本番購入や実広告視聴に見えないか。
- トップメニュー上部で残り焼きまんじゅうが串つきで5本分表示され、PLAYを押す前に残り本数が自然に目に入るか。プレイ後に1つ食べたように減り、0の時は報酬なしの練習プレイとして始まり、Gだるま/ベスト/デイリー/完走数が増えないか。
- 焼きまんじゅう0のトップメニューで「焼き上がりまで」の回復時間と「練習は報酬なし」が読め、プレイヤーが待つ/練習する/テスト回復する違いを誤解しないか。
- リザルトでスコア、ランク、見どころワード、報酬サマリーだけが短く読め、練習プレイでは報酬なしが分かるか。
- トップメニューで、直近結果と主要ボタンが先に目に入り、日課/スコア/週目標はミッション画面へ分離されてプレイ後のリアクション導線を邪魔していないか。
- リザルトの報酬サマリーで、今回増えたGだるまと達成理由が分かり、練習時は報酬なしが誤解なく伝わるか。
- リザルト後のボタンが `もう1回` / `トップへ` / `デッキ構築` の3つだけで、トップメニューの雰囲気と揃っているか。
- 設定画面やリザルトに、プレイヤーを冷めさせるフィードバック入力やテストレポート導線が残っていないか。
- 設定画面の`SOUND 音を確認`で効果音が鳴り、配信前やイヤホン接続時に音量確認として使えるか。サウンドOFF時はOFF状態が伝わるか。
- デッキ構築/コレクション画面で、カードをタップした時に読み、地域メモ、文字数/スキル/所持状態、直接アクションが自然に読めるか。
- デッキ構築/コレクション画面で、`くさつおんせん` が編成候補から外れ、スタミナの印象につながる `やきまんじゅう` がカード一覧に残るか。
- コレクションの絞り込み `ALL/所持/未所持/GGG/GG/G`、文字数フィルタ `文字ALL/3字/4字/5字+`、ソート `おすすめ/レア/名前` が、デッキ構築時の探しやすさに役立っているか。
- ことばメモに `話題/学び/配信/視聴` など開発側の狙いラベルが出ず、プレイヤー向けの情報だけで冷めずに読めるか。
- 今日のことばが、カードの読みや地域メモを確認する短い入口になっているか。
- プレイ中にPAUSE、途中終了、ホーム移動、アプリ復帰を行っても、時間が勝手に減らず再開/終了の意味が分かるか。リザルト後にトップメニューへ戻れるか。

- 1プレイ目でルールを説明なしに理解できるか。
- スライドしたい方向にパネルが動くか。
- 自動消去が「勝手に消えた」ではなく「動かしてつながったから消えた」と感じるか。
- 順番自由のルールが気持ちよさにつながっているか。
- 設定のパネル番号表示ON時に、盤面の番号が読め、かな文字を邪魔しすぎないか。デッキ色チップは番号ではなく色名と `推し/デッキ入り` の状態が読めるか。
- 消える頻度が少なすぎないか、多すぎて考える余地がなくないか。
- 3文字カードだけで固めたデッキが、コンボ頻度だけで長い言葉デッキを大きく上回りすぎないか。5文字以上の完成時に、点数の伸びが気持ちよく伝わるか。
- 消去時のワードコールで、横から見ている人も完成語と得点を読める一方、直後のスライド対象を隠していないか。
- 旧配信切替ボタン、配信モード切替、専用情報帯、配信専用リザルトが出ていないか。
- ワードコールがトップバー直下に収まり、盤面や次の操作を塞いでいないか。
- 通常リザルトだけで、スコア、ランク、見どころワード、報酬サマリー、3つの主ボタンが390px幅でも重ならず読めるか。上側が見切れていないか。
- 詰まった時にリフレッシュを使いたくなるか。
- デッキスキルを使いたくなるか。推しカードの特別感が表示で伝わるか。
- 60秒後にもう一度遊びたいと思えるか。
- デイリー目標が「また明日触る理由」になり、プレイ制限の圧に見えすぎないか。
- 今日の差し入れモーダルが、ログイン強制ではなく「開いたらちょっと嬉しい」程度に感じられ、リザルト直後のホーム操作を邪魔しないか。
- 下部バナー広告枠が、ホーム、デッキ構築、パック購入、ゲーム中の主要ボタンや盤面を隠していないか。
- 週替わり祭りが配信や親子プレイで「今週の目標」として話題にしやすく、課金しないと損に見えないか。

## 合格目安

- 初心者の初回プレイで、60秒以内に1回以上カード名が完成する。
- 3プレイ目までに、スライド操作の迷いがほぼなくなる。
- 慣れた人は60秒で3回以上完成できる。
- 平均スコアが0点付近に張り付かない。
- 画面外にはみ出し、文字の重なり、誤タップが目立たない。

## 変更判断

- 完成しなさすぎる: デッキ文字数を短くする、盤面を広げる、出現率を寄せる。
- 消えすぎる: 自動判定間隔を少し伸ばす、長いカードを増やす。
- 操作が重い: スライドしきい値を下げる、アニメーションを短くする。
- 何をしていいか分からない: スコア表示と消去演出を強める。
- 有利不利が強すぎる: スキル効果、凸補正、スコア倍率を下げる。
